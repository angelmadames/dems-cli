## Overview

This project was created using `bun init` in bun v1.0.23.
[Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.

## Setup

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run cli.ts
```
